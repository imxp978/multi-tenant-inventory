# Inventory Management System

A multi-tenant inventory management system with seperated database for each tenant.

demo: https://inventory-demo.tsaochun.com admin / demo123
demo: https://inventory-demo2.tsaochun.com admin / demo123

- **Frontend:** Angular, Angular Material, Reactive Forms, RxJS
- **Backend:** NestJS, TypeORM, JWT Authentication, RESTful API
- **Database:** MySQL, Multi-tenant DB isolation
- **Infrastructure:** GCP Compute Engine, PM2, OpenLiteSpeed, CyberPanel
